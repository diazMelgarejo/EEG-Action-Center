# EEG Action Center

This repo is the operating system for EEG repairs, IT operations, and Wearable Sensing workflows.

## Structure
- dashboards/ → Google Sheets / Excel command center
- scripts/ → automation (Apps Script)
- docs/ → SOPs, KPIs, playbooks

## Core Principles
- No backlog growth
- Track everything
- Every meeting produces actions

## Quick Start
1. Upload EEG_Command_Center.xlsx into dashboards/
2. Use KPI Tracker daily
3. Review weekly metrics
