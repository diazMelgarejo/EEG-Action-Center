# EEG Repair SOP

## Daily Flow
1. Check backlog count
2. Prioritize High impact units
3. Allocate repair slots

## Triage
- High: client impact → same day
- Medium: within 48h
- Low: batch weekly

## Repair Steps
1. Diagnose issue
2. Identify parts needed
3. Execute repair
4. Test signal quality
5. Log completion

## Logging
Every repair must include:
- Issue category
- Time spent
- Outcome

## Goal
- Reduce repeat failures
- Maintain throughput above intake
