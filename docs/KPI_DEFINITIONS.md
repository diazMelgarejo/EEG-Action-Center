# KPI Definitions

## Core Metrics

### Backlog
Total pending EEG repair units at start/end of day.

### New Repairs
Number of new units received per day.

### Completed
Number of units repaired and closed.

### Backlog Trend
Backlog End = Backlog Start + New Repairs - Completed

### Avg Repair Time
Total repair hours / completed units.

### Repeat Failure Rate
% of devices returning with same issue.

## Targets
- Backlog: decreasing weekly
- Completion rate > intake rate
- Repeat failures < 10%
