# Wearable Sensing Playbook

## Rule
Every meeting must produce at least 3 actions.

## Before Meeting
- List top 3 blockers

## During Meeting
- Capture decisions
- Clarify ownership

## After Meeting
- Log actions
- Assign owner
- Set deadline

## Tracking
Each issue must include:
- Problem
- Root cause
- Next action
- Status

## Goal
- Zero stagnant tickets
- Continuous forward movement
